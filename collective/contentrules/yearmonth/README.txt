collective.contentrules.yearmonth
=================================

This package defines a content rules action similar to the the standard move one provided by Plone.
You are able to choose a destination target folder and the element will be moved into the destination/YYYY/MM folder.
Year and month depends on the publishing date and they are automatically added if not existent.

It may be useful for particular news publishing processes.
